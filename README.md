## Installation

> Built as UserCSS to use with **Stylus**:

1️⃣ Install [Stylus browser extension](https://add0n.com/stylus.html)  
2️⃣ Copy each style’s code into a new UserStyle  
3️⃣ Save & enable – enjoy your sakura glass world! 🌸
